React 是一个 UI Library，用来把组件翻译成 DOM。流程可抽象为两个阶段：

1. **Render/Reconcile**：对比虚拟树，计算需要变更的节点与属性。
2. **Commit**：把变更应用到 DOM（或宿主），同步副作用。

常见优化方向：

- 合理拆分组件、用 `memo` 避免不必要渲染
- 使用 `useTransition`、`Suspense` 改善交互流畅性
- 列表类组件关注 key 与分页/虚拟化
